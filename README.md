## Uso

1. **Iniciar la aplicación**:
   Abre el archivo `index.html` en tu navegador.

2. **Convertir archivos**:
   - Usa el área de entrada para ingresar o cargar texto.
   - Selecciona el formato de salida (TXT o JSON) desde el menú desplegable.
   - Haz clic en "Export Result" para descargar el archivo convertido.

3. **Historial de uso**:
   El historial de conversiones se guarda automáticamente en `localStorage` y se muestra en la sección correspondiente.

4. **Tema claro/oscuro**:
   Puedes cambiar entre tema claro y oscuro utilizando el botón "Toggle Theme" en la esquina superior derecha.

## Tecnologías usadas

- HTML
- CSS (con estilos modernos y respuestas a diferentes pantallas)
- JavaScript (para manejar lógica de conversión, historial y temas)
